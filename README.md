## 使用方法

首先你需要购买游戏本体(在[DLSite](https://www.dlsite.com/maniax/work/=/product_id/RJ354175.html))

然后你会在游戏的`www/data/`目录下见到一堆json文件. 它们就是游戏的数据.

用本文件中的json替换之,就OK了.

Release 中有现成的json,这样你就不必把整个git仓都复制进去.
